Structure

Following example of VOTable was created from SDSS FITS file used in this work. First there is an information about XML and VOTable versions and references to corresponding XML Schema . <TABLE> tag encapsulating tabular data. <FIELD> tag describe identifinaction (ID), type and precision of columns. <DATA> tag contains data (here) in TABLEDATA format (other types are FITS and BINARY)
